## About This Site
 
This is a single-page interactive CV with three perspective modes:
 
| Mode | 定位 | Audience |
|------|------|----------|
| Brand & Design | 品牌視覺企劃 | Design studios, brand agencies |
| Specialty Coffee | 咖啡產業行銷 | F&B brands, coffee companies |
| MARCOM | 產品行銷企劃 | Marketing teams, product companies |
 
Each mode presents the same person through a different professional lens — same experience, different emphasis and framing.
 
---
 
## Tech Stack
 
- Pure HTML / CSS / Vanilla JS
- No frameworks, no dependencies, no backend
- Fonts via Google Fonts (Urbanist + Noto Sans TC)
- Profile photo embedded as base64 (no external image requests)
- GitHub Pages compatible — single `index.html`
